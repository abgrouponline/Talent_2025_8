SHINE Academies IT Service Proposal

Proactive IT Management & Network Support for Schools

Client Requirements Summary

  • Manage and support IT networks across multiple schools.
  • Provide proactive recommendations and improvements.
  • Support both academic and administrative IT systems.
  • Comply with legislation for IT & Network Services in schools.
  • Provide KPI tracking, performance reporting, and system development advice.
  • Assist with hardware and software procurement.
  • Deliver on-site and remote support with flexible models.

Proposed Technology Stack

Network & Infrastructure

  • Managed Wi-Fi and LAN infrastructure
  • Firewall & advanced security solutions
  • VPN for secure remote access
  • Multi-site network monitoring tools

Software & Systems

  • Cloud-based collaboration tools (Microsoft 365 / Google Workspace)
  • School management systems (SIMS, Arbor, or similar)
  • Endpoint management & antivirus solutions
  • Backup and disaster recovery systems

Support & Monitoring

  • Remote monitoring & alerting software
  • Ticketing and support management system
  • Performance dashboards for KPI tracking
  • Automated maintenance scheduling

Implementation Roadmap

Phase 1: Assessment

  • Audit current IT infrastructure across schools
  • Identify gaps, risks, and improvement opportunities
  • Define KPIs and reporting structure

Phase 2: Planning & Design

  • Design multi-site network architecture
  • Plan hardware and software upgrades
  • Define support model (on-site + remote)

Phase 3: Implementation

  • Deploy network improvements and security solutions
  • Install and configure software systems
  • Provide training for school staff

Phase 4: Monitoring & Continuous Improvement

  • Regular performance reporting to Trustees
  • Proactive system maintenance and enhancements
  • Ongoing IT support and recommendations

Expected Results

Enhanced IT Performance

Reliable, secure, and high-performing network and systems across all schools.

Proactive Improvement

Continuous recommendations for IT growth and efficiency aligned with SHINE Academies strategy.

Compliance & Reporting
